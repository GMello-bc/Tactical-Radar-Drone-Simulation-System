# Arquivos de Áudio do Jogo

Coloque os seguintes arquivos .mp3 nesta pasta para que o áudio do jogo funcione:

- **select.mp3** - Som ao selecionar botões
- **launch.mp3** - Som ao lançar drone
- **laser.mp3** - Som ao disparar laser
- **explosion.mp3** - Som de explosão
- **return.mp3** - Som ao drone retornar à base

Todos os arquivos devem estar em formato MP3 para compatibilidade com Pygame.
